[Spring] мҠӨн”„л§Ғл¶ҖнҠё + Mybatis лҚ°мқҙн„°мҶҢмҠӨ м—¬лҹ¬к°ң м—°кІ° (мҠӨн”„л§Ғ лӢӨмӨ‘ лҚ°мқҙн„°лІ мқҙмҠӨ м—°кІ°)

October 20, 2023 В· Jun Kang

лӢӨмқҢмқҖ мҠӨн”„л§Ғл¶ҖнҠё + л§Ҳмқҙл°”нӢ°мҠӨ н”„лЎңм қнҠём—җ 2к°ң мқҙмғҒмқҳ лҚ°мқҙн„°мҶҢмҠӨлҘј м—°кІ°н•ҳлҠ” мҳҲм ңмқҙлӢӨ.В 

кё°мЎҙ мҠӨн”„л§Ғ лҚ°мқҙн„°лІ мқҙмҠӨ м—°лҸҷмқҙ лҗҳм–ҙмһҲлӢӨкі  к°Җм • нӣ„ м§„н–үлҗҳл©°, мӢ к·ң лҚ°мқҙн„°лІ мқҙмҠӨлҠ” кө¬лі„мқҙ мүҪлҸ„лЎқ new лҚ°мқҙн„°лІ мқҙмҠӨлЎң лӘ…м№ӯ н•ңлӢӨ.

1. мӢ к·ң лҚ°мқҙн„°лІ мқҙмҠӨ м ‘мҶҚм •ліҙ м¶”к°Җ#

м¶”к°Җлҗ  лҚ°мқҙн„°лІ мқҙмҠӨмқҳВ  мҶҚм„ұмқ„ application.propertiesм—җ м¶”к°Җ

#кё°мЎҙ DB Setting
spring.datasource.url={url}
spring.datasource.driverClassName={driverClassName}
spring.datasource.username={username}
spring.datasource.password={password}
spring.datasource.schema={schema}

#м¶”к°Җлҗ  DB Setting
new.datasource.url={url}
new.datasource.driverClassName={driverClassName}
new.datasource.username={username}
new.datasource.password={password}
new.datasource.schema={schema}

2. лҚ°мқҙн„°лІ мқҙмҠӨ м—°кІ° л№Ҳ мҶҚм„ұ м¶”к°Җ#

кё°мЎҙ лҚ°мқҙн„°лІ мқҙмҠӨ м—°кІ° м„Өм •мқ„ н•ҳлҠ” DatabaseConfig.java мҷём—җ NewDatabaseConfig.javaмӢ к·ңлЎң м¶”к°Җн•ңлӢӨ.

кё°мЎҙ DatabaseConfig.java

@Bean(name="sqlSessionFactory")
@Primary

    - sessionFactory м„Өм • -
    return sessionFactory.getObject();

    return  new SqlSessionTemplate(sqlSessionFactory);

    - лҚ°мқҙн„°лІ мқҙмҠӨ м—°кІ°м„Өм • -

мӢ к·ң м¶”к°Җлҗң NewDatabaseConfig.java м„Өм •

лӢӨмқҢ мҳҲм ңлҢҖлЎң SqlSessionFactory, SqlSessionTemplate, DataSource л№Ҳл§Ң лі„лҸ„лЎң л“ұлЎқн•ҙ мЈјл©ҙ м„Өм •н•ң лҚ°мқҙн„°лІ мқҙмҠӨ мЈјмҶҢлЎң м—°кІ°мқҙ к°ҖлҠҘн•ҳлӢӨ. В 

лӢӨл§Ң, NewDatabaseConfig.javaмқҳ л№Ҳ м„Өм •мқ„ ліҙл©ҙВ 

public SqlSessionTemplate newSqlSessionTemplate(@Qualifier("newSqlSessionFactory") SqlSessionFactory newSqlSessionFactory)

лӢӨмқҢкіј к°ҷмқҙ мөңмҙҲ м„Өм •м—җлҸ„ SqlSessionTemplate, SqlSessionFactory л№Ҳ л“ұлЎқ мӢң мғҲлЎң мғқм„ұлҗң SqlSessionFactory, DataSourceлҘј лӘ…мӢңн•ҙмЈјм§Җ м•ҠлҠ”лӢӨл©ҙ кё°мЎҙ м„Өм •мқҳ @Primary м„Өм •мқ„ л”°лқј кё°мЎҙмӮ¬мҡ© мӨ‘мқё лҚ°мқҙн„°лІ мқҙмҠӨ м ‘мҶҚ м •ліҙлҘј мӮ¬мҡ©н•ҳкІҢ лҗҳкё°м—җ мқҳмЎҙм„ұ мЈјмһ…мқ„ лі„лҸ„лЎң н•ҙмЈјм–ҙм•ј н•ңлӢӨ. лі„лҸ„лЎң лӘ…мӢңн•ҳм§Җ м•Ҡмқ„ кІҪмҡ°, мөңмҙҲ л№Ҳ л“ұлЎқмӢң мӢ к·ң лҚ°мқҙн„°лІ мқҙмҠӨлҠ” м»Ёл„Ҙм…ҳл§Ң л§әмқҖ мұ„лЎң, мӢ к·ң лҚ°мқҙн„°лІ мқҙмҠӨ м—°кІ° м„Өм •лҸ„ 1лІҲ лҚ°мқҙн„°лІ мқҙмҠӨ м—°кІ° м„Өм • л№Ҳмқ„ м°ёмЎ°н•ҳкІҢ лҗңлӢӨ.

3. лҚ°мқҙн„°мҶҢмҠӨ м„Өм •#

м¶”к°Җ лҚ°мқҙн„°лІ мқҙмҠӨ м—°кІ° л№Ҳ м„Өм •мқҙ мҷ„лЈҢлҗҳл©ҙ л“ұлЎқлҗң 2к°ңмқҳ лҚ°мқҙн„° мҶҢмҠӨ мӨ‘ м ‘мҶҚм •ліҙлҘј м„ нғқн•ҳм—¬ мӮ¬мҡ© к°ҖлҠҘн•ҳлӢӨ.

private final SqlSessionTemplate sqlSessionTemplate;
private final SqlSessionTemplate newSqlSessionTemplate;

public TestRepository(@Qualifier("newSqlSessionTemplate") SqlSessionTemplate sqlSessionTemplate, SqlSessionTemplate sqlSessionTemplate) {
    this.newSqlSessionTemplate = newSqlSessionTemplate;
    this.sqlSessionTemplate = sqlSessionTemplate;
}

--кё°мЎҙ лҚ°мқҙн„°лІ мқҙмҠӨ м—°кІ°
public Integer connectionTest() throws Exception {
    log.info(newSqlSessionTemplate.getConnection().get~()); -- м ‘мҶҚ м •ліҙ нҷ•мқё
    return newSqlSessionTemplate.selectOne("~.connectionTest");
}
